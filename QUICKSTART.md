# Быстрый старт

## Установка и запуск

### Windows

```cmd
:: 1. Создайте виртуальное окружение
python -m venv venv

:: 2. Активируйте его
venv\Scripts\activate

:: 3. Установите зависимости
pip install -r requirements.txt

:: 4. Запустите приложение
python src/app.py
```

## Первый запуск

1. По умолчанию включен запуск в системном трее, поэтому главное окно может не открыться автоматически
2. Откройте окно через иконку в трее (двойной клик или пункт `Открыть`)
3. Вы увидите список предустановленных шаблонов комментариев
4. При необходимости измените поле горячей клавиши и флаг "Лог"
5. Нажмите "Сохранить" для создания/обновления конфигурационных файлов
6. Закройте окно (программа свернется в трей)

## Тестирование

1. Откройте любой текстовый редактор (например, Notepad++ или VS Code)
2. Напишите несколько строк кода:

```
Процедура ОбработкаДанных()
    // Код обработки
КонецПроцедуры
```

3. Выделите строку `// Код обработки`
4. Нажмите глобальную горячую клавишу (по умолчанию `Ctrl+=`)
5. В открывшемся диалоге выберите нужный комментарий
6. Текст автоматически будет модифицирован!

## Пример результата

**До:**
```
// Код обработки
```

**После применения шаблона "Код добавлен":**
``` 
// + ILYA. 20.02.2026 10:30. [PROJECT-101] Пример задачи
// Код обработки
// - ILYA. 20.02.2026 10:30.
```

## Редактирование шаблонов

1. Откройте главное окно (двойной клик по иконке в трее)
2. Выберите шаблон в таблице
3. Нажмите "Редактировать"
4. Измените название или шаблон
5. Нажмите "ОК"

## Возможные проблемы

### Windows: "Нет прав для регистрации горячих клавиш"
- Запустите приложение от имени администратора
- Или используйте другую комбинацию клавиш

### Не работает в конкретном приложении
- Некоторые приложения блокируют глобальные горячие клавиши
- Попробуйте запустить программу с правами администратора

## Завершение работы

1. Найдите иконку в системном трее
2. Правый клик -> "Выход"
3. Либо используйте пункт "Выход" в меню иконки в системном трее
